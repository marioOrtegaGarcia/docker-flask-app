# Containerized Website

## The Stack
- Docker
- Nginx
- uWSGI
- Python
  - Flask
- HTML5
  - Bootstrap 4

## Running the stack
``docker-compose up --build``

## Development
### Starting Virtual Environment
``source env/bin/activate``
